# Advent of Code 2021

Code puzzle implementations for the [Advent of Code 2021](https://adventofcode.com/2021). The puzzles are written in C# with [.NET 7](https://dotnet.microsoft.com/en-us/download/dotnet/7.0).

Each day consists of a separate console application within the Visual Studio Solution. However, you can also run a puzzle by executing the following command in the appropriate folder:

```
dotnet run
```
